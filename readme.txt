== Description ==
Simple antispam plugin without questions on Javascript-solution. Without JS-Solutions give it an textbox. The content in the textbox is configurable with own words. Also it is possibilty, configurable the options to delete spam from comment, trackback and pingback.

Please visit [the official website](http://bueltge.de/wp-js-antispam-plugin/418/ "JS AntiSpam") for further details and the latest information on this plugin.

== Installation ==
1. Unpack the download-package
1. Upload all files to the `/wp-content/plugins/` directory, without folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Got to 'Manage' menu and configure the plugin

See on [the official website](http://bueltge.de/wp-js-antispam-plugin/418/ "JS AntiSpam").
